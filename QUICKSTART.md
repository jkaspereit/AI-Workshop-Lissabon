# 🚀 Quick Start Challenges

## Get Up and Running in 5 Minutes!

This quick guide will help you start the workshop as fast as possible.

### Step 1: Install Dependencies (2 minutes)

Open your terminal in this directory and run:

```bash
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
# OR
venv\Scripts\activate     # On Windows

pip install -r requirements.txt
```

### Step 2: Start Jupyter (30 seconds)

```bash
jupyter notebook
```

This will open Jupyter in your web browser.

### Step 3: Open the Notebook (10 seconds)

Click on `01)digit_recognition_workshop.ipynb` for example, in the Jupyter interface.

### Step 4: Run the Workshop! (5 minutes)

Simply run each cell from top to bottom

- **Read the markdown cells carefully** - they explain what each code cell does
- **Don't skip cells** - each cell builds on the previous ones
- **Experiment!** - Try changing parameters to see what happens
- **Take your time** - Understanding is more important than speed
